# ALG Visualizer Site

一个可本地启动的算法可视化学习网站。

## 安装

### 全局安装

```bash
npm install -g alg-visualizer-site
```

安装后直接运行：

```bash
alg-visualizer
```

默认会在本机启动：

```text
http://localhost:4173
```

### 使用 npx 直接运行

```bash
npx alg-visualizer-site
```

### 自定义端口

```bash
alg-visualizer --port=8080
```

## 适用场景

- 本地学习算法可视化
- 给学生或同事直接分发可运行网站
- 不想进源码目录、只想直接启动站点

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 打包检查

```bash
npm run pack:check
```
